Introduction¶

For this project, The Movie Database (TMDB) dataset will be investigated for insights and relationship. TMDB is a popular user editable database for movies and TV shows. It contains information and statistics about movies, actors, directors as well as other film industry professionals. This dataset has movies details such as revenue, budget, genres, popularity, etc. all contained in it.

Questions for Analysis¶

  • Which movie has the minimum and maximum Budget?
  • Which movie generate the minimum and maximum Revenue?
  • Which movie generate the minimum and maximum Profit?
  • Which movie has the highest and lowest Popularity Score?
  • Which movie has the highest and lowest Vote Average?
  • Which is the most produced Genres?
  • What is the budget over the Years?
  • What is the revenue over the Years?
  • Relationship within the dataset
    • Budget & Revenue.
    • Revenue & Profit.
    • Popularity & Runtime.

    • Profit & Vote Average.

The said function would return a count of values in that column.

In [66]:
# Defining our function
def Val_count(colm):
    
    # joining the content of the column together with pipe "|" as a seperator
    join = mov_df[colm].str.cat(sep='|')
    
    # turning the above to a pandas series
    series = pd.Series(join.split('|'))
    
    # counting the value of the series
    count = series.value_counts()
    
    # returns the counts as result
    return count

Movie with minimum and maximum Budget¶

Let's create a function that will return a dataframe of maximum and minimum row of an int column concatenated together. This function will be used across different columns for details.

In [70]:
# Defining our function
def max_min(colm):
    
    # storing max id
    idMax = mov_df[colm].idxmax()
    
    # storing min id
    idMin = mov_df[colm].idxmin() 
    
     # bringing out ths max row
    Max = pd.DataFrame(mov_df.loc[idMax])
    
    # bringing out the min row
    Min = pd.DataFrame(mov_df.loc[idMin]) 
    
    # concatenating both row
    bth_row = pd.concat([Max,Min], axis =1) 
    
    # returning both as result
    return bth_row

Limitation of the analysis
